主要特性：

评测机可部署在多台不共享文件系统的机器上

对异构多核可以将评测机分组并限制在对应核心

数据库中的数据对象。

所有请求和返回值都用 json

分为 Controller 和 Service 模块。其中 Controller 模块根据对应功能调用 Service 中的函数，并依此来判断 http 返回状态与内容。

Service 模块执行鉴权、实际的业务逻辑。
